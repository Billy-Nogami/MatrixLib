# MatrixLib

Учебная библиотека на C для работы с матрицами.

## Возможности

- создание и освобождение матриц
- чтение и запись элементов
- заполнение матрицы значением
- копирование матриц
- сложение и вычитание
- умножение на скаляр
- умножение матриц
- транспонирование
- вычисление определителя

## Сборка и тесты

```sh
make
make test
```

## Структура

- `include/matrix.h` — интерфейс библиотеки
- `src/` — реализация модулей
- `tests/` — тесты
