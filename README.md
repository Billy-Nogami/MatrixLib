# MatrixLib

Учебная библиотека на C для работы с матрицами.

## Описание

Проект представляет собой модульную библиотеку на языке C для выполнения базовых операций линейной алгебры над матрицами. Библиотека разработана как учебная работа с акцентом на:

- разбиение программы на отдельные модули
- использование динамической памяти
- работу со структурами и указателями
- обработку ошибок через коды возврата
- тестирование реализованных функций

Матрица хранится в структуре `Matrix`:

```c
typedef struct {
    size_t rows;
    size_t cols;
    double *data;
} Matrix;
```

Элементы матрицы располагаются в одном непрерывном участке памяти, выделяемом динамически. Это упрощает освобождение памяти и делает работу с данными более удобной, чем при использовании `double **`.

## Реализованные возможности

- создание и освобождение матриц
- чтение и запись элементов
- заполнение матрицы значением
- копирование матриц
- сложение и вычитание
- умножение на скаляр
- умножение матриц
- транспонирование
- вычисление определителя

## Функции библиотеки

- `matrix_create` — выделение памяти под матрицу заданного размера
- `matrix_free` — освобождение памяти и очистка структуры
- `matrix_set` — запись значения в элемент матрицы
- `matrix_get` — чтение значения элемента матрицы
- `matrix_fill` — заполнение всех элементов одинаковым значением
- `matrix_copy` — создание копии матрицы
- `matrix_add` — сложение двух матриц одинакового размера
- `matrix_sub` — вычитание двух матриц одинакового размера
- `matrix_scalar_mul` — умножение матрицы на число
- `matrix_mul` — умножение двух матриц
- `matrix_transpose` — транспонирование матрицы
- `matrix_determinant` — вычисление определителя квадратной матрицы

## Обработка ошибок

Большинство функций возвращают значение типа `MatrixStatus`. Используются следующие коды:

- `MATRIX_SUCCESS` — операция выполнена успешно
- `MATRIX_ERROR_NULL` — передан `NULL` или матрица не инициализирована
- `MATRIX_ERROR_ALLOC` — ошибка выделения памяти
- `MATRIX_ERROR_BOUNDS` — выход за границы матрицы
- `MATRIX_ERROR_SIZE` — несовместимые размеры матриц
- `MATRIX_ERROR_NOT_SQUARE` — матрица не является квадратной

## Структура проекта

- `include/matrix.h` — описание структуры `Matrix`, кодов ошибок и интерфейса библиотеки
- `src/matrix_memory.c` — создание, освобождение и копирование матриц
- `src/matrix_access.c` — доступ к элементам и заполнение
- `src/matrix_basic.c` — базовые арифметические операции
- `src/matrix_advanced.c` — умножение, транспонирование и определитель
- `tests/test_matrix.c` — набор тестов для проверки работы библиотеки
- `Makefile` — сборка проекта и запуск тестов

## Пример использования

```c
#include <stdio.h>
#include "matrix.h"

int main(void) {
    Matrix a = {0};
    Matrix b = {0};
    Matrix result = {0};

    if (matrix_create(&a, 2, 2) != MATRIX_SUCCESS) {
        return 1;
    }

    if (matrix_create(&b, 2, 2) != MATRIX_SUCCESS) {
        matrix_free(&a);
        return 1;
    }

    matrix_set(&a, 0, 0, 1.0);
    matrix_set(&a, 0, 1, 2.0);
    matrix_set(&a, 1, 0, 3.0);
    matrix_set(&a, 1, 1, 4.0);

    matrix_fill(&b, 1.0);

    if (matrix_add(&a, &b, &result) == MATRIX_SUCCESS) {
        double value = 0.0;
        matrix_get(&result, 1, 1, &value);
        printf("result[1][1] = %.2f\n", value);
    }

    matrix_free(&a);
    matrix_free(&b);
    matrix_free(&result);
    return 0;
}
```
